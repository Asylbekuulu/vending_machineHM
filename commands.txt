В данной домашке были использованы следующие команды из GIT:
git init - для инициализации Гита в локальном репозитории
git add - с помощью данной команды я добавлял изменненые файлы в отслеживания для дальнейшего коммита
git commit -m "" - с помощью жтой команды я коммитил мои изменения в проекте
git reset --hard - возвращался к предыдущему коммиту с удалением всех последних изменений
git checkout <имя файла> - перемещялся между ветками Master и home_work_25
git merge <имя ветки> - мержил ветку home-work_25 в главную ветку Master
git remote add origin https://github.com/Asylbekuulu/HomeWork25.git - добавил удаленный репозиторий
git push origin Master - запушил домашку на удаленный репозиторий
